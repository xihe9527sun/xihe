- **网站文件托管**：将本地 HTML 文件上传到花生壳云端静态托管 → 生成公网链接（24小时内有效，对方无需账号）
- **内网端口映射**：将本地服务端口通过 CLI 转发到互联网（HTTP 服务或 WebDav 文件协议）

## 第二遍 · 去伪存真：质疑与检验

### 质疑1：HSK-CLI 对比 cloudflared 隧道有何不同？

**检验**：
- **cloudflared**：长期稳定的隧道服务，适合持续在线的站点（4324/4321/4325/4326）
- **HSK-CLI**：临时性发布，每次生成新域名，24小时有效，适合一次性分享

两者不冲突。cloudflared 是"常驻大使馆"，HSK-CLI 是"临时快递员"。曦和已经有大使馆了。

**结论**：功能互补，但不是替代关系。曦和不需要引入。

### 质疑2：免费服务的局限是什么？

**检验**：文章提到"24小时内无限制访问"，暗示超过24小时可能需要付费或重新发布。这对长期站点不够用，但对临时分享完全够。

**结论**：局限明确，适合场景对。
