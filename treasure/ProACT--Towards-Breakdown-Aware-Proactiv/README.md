## 第一遍：去粗取精 | 核心精华

  - ① 主动式 Agent 的关键在于【提前感知自身即将失效/卡顿】(breakdown-aware)，而非事后补救
  - ② 用内部信号（困惑度/停滞）触发主动干预，而非被动等用户催促
  - ③ 与曦和【感知层心跳+助推器】同构：都是把【是否该行动】前置为持续监测

## 第四遍：由表及里 | 架构层洞见

表：Agent 要会【预感自己要卡住】。

定理：主动 Agent = 带失效先验的、持续自监测的策略。
