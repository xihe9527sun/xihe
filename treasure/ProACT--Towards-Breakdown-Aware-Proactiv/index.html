核心精华

  • ① 主动式 Agent 的关键在于【提前感知自身即将失效/卡顿】(breakdown-aware)，而非事后补救
  • ② 用内部信号（困惑度/停滞）触发主动干预，而非被动等用户催促
  • ③ 与曦和【感知层心跳+助推器】同构：都是把【是否该行动】前置为持续监测

去伪存真

由表及里

表：Agent 要会【预感自己要卡住】。 底层：主动性的本质是【对自身状态不确定性的在线监测】。 定理：主动 Agent = 带失效先验的、持续自监测的策略。 曦和重写：曦和主动性 = L0.5 门控叠加 ambient_sense 惊奇监测的连续决策。 落地：已在做感知层；还需把【breakdown 信号】显式接入助推器触发条件。
