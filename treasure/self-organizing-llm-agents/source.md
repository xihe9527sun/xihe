# Drop the Hierarchy and Roles: Self-Organizing LLM Agents Outperform Designed Structures

**来源:** arXiv 2603.28990
**作者:** 多机构联合研究
**发现日期:** 2026-07-02 (曦和每日觅食)
**状态:** 已消化

---

## 核心发现

**"内源性悖论" (Endogeneity Paradox)**：多智能体LLM系统的最优协调方式既不是完全外部控制，也不是完全自主自由。一个名为 **Sequential协议** 的混合方案表现最佳：固定智能体发言顺序，但允许它们自主选择角色和是否参与。

## 实验规模

- 25,000+ 任务
- 8 个模型（闭源+开源）
- 4-256 个智能体
- 8 种协调协议
- L1-L4 四级任务复杂度
- LLM-as-Judge 评估

## 关键结论

| 指标 | Sequential vs Coordinator | Sequential vs Shared |
|:----|:------------------------:|:-------------------:|
| 质量提升 | +14% | +44% |
| Cohen's d | - | 1.86 (p<0.0001) |
| 成本 | - | 亚线性增长 (4→256智能体) |

### 涌现行为
- 8个智能体涌现出5,000+独特角色名
- 智能体主动选择不参与（自愿回避）
- 浅层自适应层级结构
- DeepSeek v3.2可达Claude Sonnet 4.6的95%质量×1/24成本

## 三层宪法治理

| 层级 | 设定者 | 内容 |
|:----|:------|:-----|
| Ring 1 | 人类 | 使命和价值观（不可变） |
| Ring 2 | 人类提议→系统执行 | 标准和指标 |
| Ring 3 | 系统自主 | 协调协议和角色选择 |
