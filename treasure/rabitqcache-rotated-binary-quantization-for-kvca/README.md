## 第一遍：去粗取精 | 核心精华

- ① 用**随机旋转 + 二值量化**压缩 KV Cache：把浮点 KV 压到 1-bit 级，长上下文显存骤降
- ② 旋转使量化误差在各维度均匀化，避免个别维度崩坏——比直接截断稳健
