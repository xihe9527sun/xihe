◆ 去粗取精 · 核心精华

▸ 芯片架构

  • **制程**：台积电 6nm
  • **核心**：140个张量处理器 + RISC-V 处理器
  • **内存**：GDDR6 DRAM + 片上SRAM（不用昂贵的HBM）
  • **互联**：每个芯片带以太网接口，全部冗余连接
  • **扩展**：8芯片/板 → 4板/盒 → 32芯片 = 1TB DRAM + 16TB带宽
  • **训练集群**：2000个芯片 = 超过一百万个RISC-V处理器协同

    • ▸ 核心理念

    1. **不用HBM**：太贵太复杂。用普通DRAM + 更多小芯片替代

    2. **开放一切**：RISC-V开源架构 + 完整开源软件栈 + 开放小芯片架构

    3. **模块化积木**：小芯片（chiplets）设计，CPU/内存/AI各自独立迭代

    4. **更小更便宜**：制造更多、更小、更便宜的芯片，不是更大更贵

    5. **冗余可靠**：所有连接都冗余，单根线缆故障不影响运行

    ▸ 成本

  • Black Hole 芯片耗资约8000万美元
  • 每次升级需数千万美元
  • 对比NVIDIA：同等算力显著更低

    • ◆ 去伪存真 · 质疑与验证

    **质疑：8000万美元叫"便宜"？**

    → 跟NVIDIA的H100/B200比确实便宜（单芯片研发成本），而且Jim Keller说的是"让更多人能建造AI系统"，不是"免费"。

    **质疑：6nm vs NVIDIA的4nm/3nm？**

    → 不用最新制程，用成熟制程+架构创新取胜——这正是τ缩放理论的核心思想：不是靠工艺微缩，是靠架构创新。

    **质疑：不用HBM性能够吗？**

    → 用更多芯片数量 + 分布式架构弥补单芯片带宽。2000个芯片协同就是答案。
