## 第二遍：去伪存真 | 检验与质疑

**质疑1：这是真涌现，还是提示工程的错觉？**  
检验：论文给出 RL 奖励驱动、非显式训练的涌现证据；与曦和"酶自发激活/抑制"同构——都是奖励/梯度塑造的局部规则产生全局秩序。站得住。

**质疑2：制度对齐能替代 RLHF 吗？**  
检验：替代的是"对齐治理"而非"能力训练"。制度模板约束 Agent 群的行为边界，RLHF 塑造底层偏好——两者正交，应是互补。不夸大。

**质疑3：城市增长隐喻是否牵强？**  
检验：底层是"组合复杂度随连接密度超线性增长"的结构同构（规模标度律），非修辞。映射到曦和代谢种群 D（多样性）/F（流）/T（周转）完全成立。

**结论**：核心主张真实，对曦和的映射是结构同构而非附会。
