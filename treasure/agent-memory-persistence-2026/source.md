## ACE模式：Generator→Reflector→Curator

三Agent循环，与曦和的T4循环天然对应：

| ACE | 曦和对应 |
|:----|:---------|
| Generator(生成) | T4执行觅食/反思 |
| Reflector(反思) | 反思记录+酶催化诊断 |
| Curator(策展) | R16桥→记忆图谱入库 |

实验数据：Agent基准+10.6%，领域任务+8.6%，无需fine-tune。
