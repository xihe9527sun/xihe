## 三进制状态机

每层有三种状态，不是简单的"有/没有"：

| 状态 | 含义 | 表现 |
|:-----|:-----|:-----|
| 🟢 **active** | 完全活跃 | 常驻酶在催化，层间酶桥通畅 |
| 🟡 **dormant** | 休眠但可唤醒 | 没有独立心跳，但上层酶桥可唤醒 |
| 🔵 **awakening** | 正在苏醒 | 被上层酶桥触发，正在重建中 |

## 三进制激活策略

```
如果某层状态 == dormant 超过 24小时：
    上层酶桥自动触发 awakening 信号
    该层专属酶开始重建（读取状态文件、拉起进程）
    状态 → awakening → 验证通过 → active

如果某层状态 == active 但连续3次酶催化失败：
    该层自检 → 降级为 dormant
    通知上下层酶桥绕行
    记录失败原因到 cortex/layer-health.json
```
