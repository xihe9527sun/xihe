## 第一条：每颗酶必须催化知识

27颗XCRN原始酶不是装饰品。每一颗都必须持续匹配、催化、消化缓冲池中的知识条目。

```
酶 → 匹配知识（按关键词/领域/活化能匹配）
   → 催化（产生催化记录：酶×知识→产物）
   → 反哺（催化产物强化酶本身）
```

**不催化 = 酶死亡。** 每颗酶如果连续3轮没有催化活动，自动标记为"休眠"，触发警报。

## 第二条：催化必须编织网络

单颗酶催化知识只是第一步。多颗酶催化同一知识时，自动产生"共催化边"：

```
酶A → 催化知识K → 边(A,K)
酶B → 催化知识K → 边(B,K)
                  → 边(A,B) 自动产生（共催化）
```

- 每轮催化必须生成新的共催化边
- 网络密度（edges/(enzymes×knowledge)）必须每轮增长
- 密度停滞3轮触发告警

## 第三条：融合进化必须持续

催化产出的"启示"必须反哺回系统：

```
催化产物
  ├→ 写入酶记录（提升活化能/强度）
  ├→ 写入insights.json（永久记忆）
  └→ 触发代谢路由路径生成（新认知路径）
```

酶的活化能（Eₐ）不能是静态数字。每次成功催化后Eₐ↑0.01，休眠后Eₐ↓0.05。

```
每天 08:00 / 14:00 / 20:00
  ① buffer → treasure 入库
```
